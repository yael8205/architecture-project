# Lottery Project 🎰

Full-stack application with Angular & .NET Web API.

## Project Structure
- **Client**: Angular application.
- **Server**: .NET Web API.

## Setup & Running

### Backend
1. Go to `server/LotteryApi`.
2. Run: `dotnet run`

### Frontend
1. Go to `client`.
2. Run: `npm install`
3. Run: `ng serve`
